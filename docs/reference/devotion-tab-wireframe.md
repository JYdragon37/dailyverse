# 묵상 앱 와이어프레임 — 묵상 탭 플로우 스펙

## 프로젝트 개요

- **디바이스**: iOS iPhone 15 Pro (393 x 852 pt, @3x)
- **Safe Area**: 상단 59pt (Dynamic Island), 하단 34pt (Home Indicator)
- **디자인 톤**: 따뜻하고 미니멀. 다크 모드 우선. 부드러운 라운드 카드 UI.
- **폰트**: Pretendard (한글), SF Pro (시스템)
- **컬러 팔레트**:
  - Primary: `#F5E6CA` (따뜻한 아이보리)
  - Accent: `#D4A574` (골드 브라운)
  - Background: `#1A1A2E` (딥 네이비)
  - Text Primary: `#FFFFFF`
  - Text Secondary: `#A0A0B0`
  - Streak Active: `#FF8C42` (주황)
  - Success: `#4CAF50`

---

## 네비게이션 구조

하단 탭바 5개:

| 순서 | 아이콘 | 라벨 | 설명 |
|------|--------|------|------|
| 1 | 🏠 | 홈 | 메인 대시보드 |
| 2 | 🔔 | 알람 | 알림 설정 및 리스트 |
| 3 | 📖 | 말씀들 | 성경 말씀 아카이브 |
| 4 | 🙏 | 묵상 | **묵상 플로우 (이 문서의 범위)** |
| 5 | 👤 | 프로필 | 유저 정보 및 설정 |

- 현재 활성 탭은 Accent 컬러(`#D4A574`)로 하이라이트
- 탭바 높이: 83pt (49pt 탭바 + 34pt Home Indicator)

### 인터랙션

| 요소 | 액션 | 결과 |
|------|------|------|
| ← 뒤로 | 탭 | Screen 1로 pop |
| 🔊 음성으로 듣기 | 탭 | 오디오 재생 시작, 버튼이 ⏸ 일시정지로 변경 |

### 스크롤 동작 참고

- CTA "오늘의 묵상 완료하기" 버튼은 **하단 고정(sticky bottom)**
- 유저가 해석을 끝까지 읽지 않아도 버튼은 항상 접근 가능
- 단, 첫 진입 시 버튼 opacity 50% → 3초 후 100%로 전환 (최소 읽기 시간 유도)

### CTA 활성 조건

"오늘의 묵상 마치기" 버튼은 아래 조건이 **모두** 충족될 때 활성화:

1. 읽기 섹션: `isCompleted === true` (녹음 완료 or 타이핑 완료)
2. 기도 섹션: `prayer.input.length > 0`

미충족 시: 버튼 opacity 40%, 탭 시 미완료 섹션으로 스크롤 + 하이라이트

### 스트릭 계산 로직

```
streak_rules:
  - 기준 시간: 매일 00:00 ~ 23:59 (유저 로컬 타임존)
  - 연속 조건: 전일에 completedAt 기록 존재
  - 끊김 조건: 전일 기록 없음 → currentStreak = 1로 리셋
  - 복구 없음: 한번 끊기면 1부터 재시작 (그레이스 피리어드 없음)
  - longestStreak: max(currentStreak, longestStreak) 자동 갱신
```
